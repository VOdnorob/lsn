Running the Project
Clone the repository or download the source code.
Open the project in your preferred code editor or IDE.
Ensure the package structure is correct
Run the Main class to execute all tasks sequentially.
Usage
Upon running the Main class, the program will prompt you to provide inputs for each task. Follow the on-screen instructions:

Task 1: Enter integers separated by spaces when prompted.
Task 2: Enter integers in one line separated by spaces when prompted.
Task 3: First enter the number of edges, followed by each edge described by two integers (representing connected vertices).
